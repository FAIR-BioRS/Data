# FAIR-BioRS-data
This is the dataset associated with our review manuscript on FAIR Biomedical Research Software (FAIR-BioRS) guidelines [1]. It is structured according to the SPARC Data Structure v2.0 [2,3] and curated using the SPARC data curation software SODA v5.3.3 [4,5]. We refer to the manuscript for details about the dataset content.
